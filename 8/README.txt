Implement one feeding for a player from the dealer.

compile - Compiles the project so it may be run.

xstep - Read a dealer configuration and apply one eating for the current
        player, then print the new dealer configuration.

dealer/evolution/evolution-server/src/dealer/mod.rs - The implementation
    of a dealer, which is obtained by deserializing a configuration.

dealer/evolution/src/wire/configuration/mod.rs - The wire configuration type.

dealer/evolution/game/game/mod.rs - Contains a new owned game datatype.
dealer/evolution/game/player/mod.rs - Contains a new owned player datatype.

rest.txt - The rest of the player interface, see supporting data structures
    inside the evolution directory.
